# MaxSupportBot
Бот, с веб-интерфейсом для операторов, для удобной поддержки пользователей в мессенджере MAX.

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка

Создайте файл `.env`:

```env
BOT_TOKEN=ВАШ_ТОКЕН_MAX
WEB_PASSWORD=ВАШ_ПАРОЛЬ
WEB_HOST=0.0.0.0
WEB_PORT=8080
```

### 3. Запуск

```bash
python main.py
```

Веб-интерфейс будет доступен по адресу: `http://localhost:8080`
